Do Loop

The do loop is a control structure similar to the while loop. It differs only on one point: a do loop always executes its body at least once, and it starts testing whether it should stop only after that first execution. To reflect this, the test appears after the body of the loop: 

            
          do {
            var name = prompt("Who are you?");
          } while (!name);
          console.log(name);

Version 1 - "MVP"

I wanted to get something up to show and this was the fastest thing I thought of. But it's a procedural approach and is not possible to change.


  var h = ['#', '# #', '# # #', '# # # #',
          '# # # # #', '# # # # # #', '# # # # # # #' ];

  var $t = $('.triangle-wrapper-1');

  _.each(h, function(hash){
    $t.append("<p>" + hash + "</p>");
  });

Pros

  • Fast
  • Easy

Cons

  • Static

Version 2 - Use objects with flexible quantity

Next approach is to use JS objects to manage creation of the glob.


    function Hash(qty) {
      this.quantity = qty;
    }

    Hash.prototype = {
      getQty: function() {
        var str = '';
        var $t = $('.triangle-wrapper-2');
        _.times(this.quantity, function(){
          str = str + "# ";
          $t.append("<p>" + str.replace(/ $/, "") + "</p>");
        });
      }
    };
    var hash = new Hash(7);
    hash.getQty();
